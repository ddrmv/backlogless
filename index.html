    Help & Shortcuts

    • ctrl+enter: New task below
    • alt+enter: New task above
    • up/down (shift+p/shift+n): Move selection
    • alt+up/down: Move task up/down

    • enter/f2: Edit task
    • ctrl+left/right: Unindent/Indent
    • shift+left/right: Mark active/completed/remove
    • shift+enter: Complete and remove

    • ctrl+i: Toggle header image
    • ctrl+m: Toggle settings menu
    • shift+t: Toggle theme
